Reinforcement Learning Algorithm Comparison

Sparse Reward Environment

Environment Setup

A 3x3 grid world where 70% of states have zero reward. This environment is designed to test each algorithm's ability to explore efficiently and discover sparse rewards. The agent must navigate from a fixed start state to a goal state, but most intermediate states provide no feedback, making exploration crucial.

Performance Analysis

Q-Learning typically excels in this environment due to its off-policy nature, which allows for more aggressive exploration while maintaining a greedy policy. SARSA tends to be more conservative in its exploration, while Monte Carlo ES can struggle with the lack of intermediate feedback.

Q-Learning

Benefits from off-policy learning, allowing separate exploration and exploitation strategies. Can learn optimal paths even from exploratory actions.

SARSA

More conservative exploration due to on-policy learning. Takes longer to find optimal paths but generally more stable once found.

Monte Carlo ES

Uses complete episode returns, which can be sparse in this environment. Exploring starts help with initial exploration but delayed feedback is challenging.

Long Horizon Environment

Environment Setup

A 3x3 grid with strategically placed checkpoints that provide partial rewards. This environment tests each algorithm's ability to handle temporal credit assignment and learn from delayed rewards. The agent must learn which checkpoint sequence leads to the optimal path.

Performance Analysis

Temporal difference methods (Q-Learning and SARSA) handle credit assignment effectively through bootstrapping. Monte Carlo methods face challenges with long-term dependencies but benefit from complete episode evaluation.

Q-Learning

Effective at propagating rewards through bootstrapping. Can quickly learn the value of checkpoint sequences.

SARSA

Similar bootstrapping benefits to Q-Learning, but with more conservative policy updates. Good stability in learning checkpoint values.

Monte Carlo ES

Evaluates complete episodes, which can be beneficial for understanding the full checkpoint sequence but slower to learn.

Stochastic Environment

Environment Setup

A 3x3 grid with noisy rewards (Ïƒ=0.2) and random delays (p=0.1). This environment tests each algorithm's robustness to uncertainty and ability to learn stable policies under stochastic conditions. The agent must learn to distinguish signal from noise.

Performance Analysis

TD methods show strong performance due to their ability to average out noise through bootstrapping. Monte Carlo methods benefit from full episode evaluation but can be more sensitive to reward variance.

Q-Learning

Robust to noise through TD learning. Can maintain good performance despite stochastic rewards and delays.

SARSA

Similar noise robustness to Q-Learning. On-policy nature can provide more stable learning in stochastic environments.

Monte Carlo ES

Full episode evaluation helps average out noise, but can be more sensitive to reward delays and variance.
